oneVita

Um pouco mais sobre o aplicativo

O aplicativo oneVita é para facilitar aos vendedores passarem pedidos para empresa oneVita. Tem layouts dinâmicos e simples, inicialmente a tela de login/cadastro do vendedor, ao efetuar o login temos a tela principal com quatro botões. Cadastro cliente, onde será registrado razão social, nome fantasia, CNPJ, IE, endereço, e-mail e o contato. Lista cliente, onde o vendedor terá a lista de seus clientes já cadastrados com suas informações. Ranking, mostra qual vendedor está vendendo mais. Estoque, mostra a quantidade de mercadoria que ainda tem para venda. Tudo será armazenado no banco de dados Firebase e passado para uma planilha no Excel para ter controle. Usamos como base o aplicativo PedidoOK, que segue a mesma linha de raciocínio de nossa ideia para o aplicativo.
